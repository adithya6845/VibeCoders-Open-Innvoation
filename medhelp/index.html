Built at the intersection of emergency medicine, artificial intelligence, and real-time simulation — aidAR gives you an unflinching, full-sensory tool for practicing life-saving skills.

aidAR Platform

Features

A comprehensive breakdown of the engine capabilities.

Procedural 3D Simulations

Perform CPR, hemorrhage control, and airway management on a real-time 3D anatomical model rendered via Three.js WebGL. Every procedure triggers a full geometry shader timeline calculated on the GPU.

Three.js WebGL

AI Medical Diagnostic Engine

The AI Doctor interprets loose, natural-language symptom descriptions and routes them to the correct First-Aid protocol. Critical conditions — stroke, cardiac arrest, anaphylaxis — get priority escalation.

Gemini LLM NLP

Spatial AR Narrator

An always-watching AR tutor that monitors the 3D camera state and selected procedure, injecting real-time contextual voice guidance at every step. Never lose track of what to do next in a crisis.

WebSpeech API

Skeuomorphic Control Console

Every element is engineered to feel physically real. Metallic gradients, inset LCD panels, drilled screws, magnetic hover physics, and glowing status LEDs — the entire interface is a tactile experience.

GSAP Vengeance UI

Physics Particle Background

A custom-authored canvas particle engine renders ambient floating motes that scatter from your cursor in real-time using vector repulsion math. Adapts seamlessly to Light and Dark modes.

HTML5 Canvas

Adaptive Lighting Themes

Full dual-mode theming — switch from burnished Aluminum Light to Stealth Black Titanium Dark at any point, even mid-session inside the app experience. Every variable transitions simultaneously.

CSS Variables

Multiplayer Trauma Scenarios

Link up with multiple medical professionals in the same instance via low-latency synchronization protocols. Practice squad-based triage and assigned procedure execution in perfect harmony.

WebRTC Data

VR & XR Headset Ready

Break out of the browser. Push a button and launch the 3D viewer directly into a connected Meta Quest or Apple Vision Pro headset using native browser APIs for full 6DOF spatial immersion.

WebXR API

Dynamic Vital Signs Engine

The patient model is not static. Blood pressure, heart rate, and oxygen saturation degrade or improve algorithmically based on the accuracy and speed of your procedural interventions.

Logic Engine
